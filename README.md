# Excel Route Manager

Настольное приложение для обработки маршрутных XLS файлов.

---

## Требования

- Python 3.10 или выше
- Windows 10/11 или Linux

---

## Установка зависимостей

```bash
pip install PyQt6 xlrd xlwt
```

---

## Запуск

```bash
python app.py
```

---

## Структура проекта

```
excel_route_py/
├── app.py                          # Точка входа
├── core/
│   ├── data_store.py               # JSON хранилище данных

│   └── excel_generator.py          # Генератор XLS файлов

│   ├── main_window.py              # Главное окно, меню, стили
│   └── pages/
│       ├── home_page.py            # Шаги 1-3: тип, файлы, папка
│       ├── departments_page.py     # Управление отделами и продуктами
│       ├── templates_page.py       # Настройка шаблонов столбцов
│       ├── settings_page.py        # Настройки отображения в штуках
│       ├── preview_general_page.py # Предпросмотр общих маршрутов
│       └── preview_dept_page.py    # Предпросмотр файлов по отделам
```

Исполняемый файл появится в папке `dist/`.

## Функционал

### Обработка файлов (Главная страница)
1. **Шаг 1** — Выбор типа файла: «Основной» или «Увеличение (Довоз)»
2. **Шаг 2** — Выбор одного или нескольких .xls файлов (перетаскивание или диалог)
3. **Шаг 3** — Выбор папки сохранения (по умолчанию — Рабочий стол)
4. Нажать «Обработать» → переход к предпросмотру

### Предпросмотр — Общие маршруты
- Таблица всех маршрутов с продуктами
- **Поиск** по адресу/номеру маршрута
- **Фильтрация** по продукту (компактный режим / маршруты полностью)
- **Редактирование** номера маршрута (двойной клик)
- **Исключение** маршрутов из итогового файла
- **Валидация** — проверка номеров маршрутов перед созданием файла
- Создание файла «Маршруты общие ДД.ММ.ГГГГ»

### Предпросмотр — По отделам
- Таблица маршрутов сгруппированных по отделам/подотделам
- Создание отдельного файла для каждого отдела/подотдела

### Отделы и продукты (меню Справочники)
- Создание отделов и подотделов (один уровень вложенности)
- Привязка продуктов к отделам/подотделам
- Каждый продукт принадлежит только одному отделу

### Шаблоны (меню Справочники)
- 2 настраиваемых шаблона для файлов по отделам
- Выбор столбцов из списка: номер маршрута, адрес, продукт, ед. изм., количество, шт
- Изменение порядка столбцов перетаскиванием
- Привязка шаблона к отделу

### Настройки Шт (меню Настройки)
- Включение/выключение отображения в штуках для каждого продукта
- Настройка «количество в 1 шт» и направления округления
- Все настройки сохраняются в JSON

---

## Логика расчёта штук

**Округление в большую сторону:**
- K = 2.7, pcsPerUnit = 1.5
- 2.7 / 1.5 = 1 шт (целая часть)
- остаток = 2.7 % 1.5 = 1.2 → 1.2 > 0.5 → +1 шт
- Итого: **2 шт**

**Округление в меньшую сторону:**
- K = 3.3, pcsPerUnit = 1.5
- 3.3 / 1.5 = 2 шт (целая часть)
- остаток = 3.3 % 1.5 = 0.3 → 0.3 < 0.5 → +0 шт
- Итого: **2 шт**

## Формат создаваемых файлов

### Общие маршруты
- Имя: `Маршруты общие ДД.ММ.ГГГГ ОС.xls` (ОС = Основной, УВ = Увеличение)
- Один лист на каждый маршрут (имя листа = номер маршрута)
- Строка 1: дата + тип файла
- Строка 2: заголовки (Номер маршрута, Адрес, Ед. изм., Количество, Шт)
- Далее: строки продуктов

### По отделам
- Имя: `Маршруты [название отдела] ДД.ММ.ГГГГ ОС.xls`
- Один лист на файл (имя листа = название отдела/подотдела)
- Маршруты отсортированы по возрастанию номера
- Строка 1: заголовок
- Далее: маршруты с продуктами
