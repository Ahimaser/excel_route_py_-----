# Варианты обновления окна «Отделы и продукты»

## Текущее состояние

Модальный диалог с деревом (QTreeWidget):
- Колонки: Название | Тип | Ед. изм. / кол-во | Печатать этикетки
- Кнопки: + Добавить отдел, + Добавить подотдел, + Привязать продукт
- Поиск по дереву
- ПКМ — контекстное меню (переименовать, удалить, вырезать/вставить)
- Drag & Drop продуктов между отделами
- ToggleSwitch для «Печатать этикетки» у отделов/подотделов

**Проблемы**: при большом числе продуктов дерево перегружено; колонка «Тип» избыточна; мало визуального разделения уровней.

---

## Вариант 1 — Split-панель: отделы слева, продукты справа

Левая панель — дерево отделов/подотделов. Правая — таблица продуктов выбранного отдела.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ Отделы и продукты                                              [ℹ] [Сохранить] │
├──────────────────────────────┬──────────────────────────────────────────────────┤
│ [🔍 Поиск...]                │ Продукты: Молоко → Чищенка                       │
│                              │                                                  │
│ ▼ 📁 Молоко (12)             │ ┌──────────────┬────────┬────────┬──────────────┐ │
│ │  📂 Чищенка (5)            │ │ Продукт      │ Ед.изм.│ Шт     │ Действия     │ │
│ │  📂 Сыпучка (3)            │ ├──────────────┼────────┼────────┼──────────────┤ │
│ │  📄 Молоко 3.2% (отдел)    │ │ Молоко 3.2%  │ л      │ ☑      │ [↑] [↓] [✕]  │ │
│ ▶ 📁 Овощи (8)               │ │ Кефир 1%     │ л      │ ☑      │ [↑] [↓] [✕]  │ │
│ ▶ 📁 Мясо (4)                │ │ ...          │        │        │              │ │
│                              │ └──────────────┴────────┴────────┴──────────────┘ │
│ [+ Отдел] [+ Подотдел]       │ [+ Добавить продукт] [Привязать из справочника]   │
└──────────────────────────────┴──────────────────────────────────────────────────┘
```

**Плюсы**: чёткое разделение, продукты в таблице — сортировка, фильтр.  
**Минусы**: два клика для переключения отдела; больше кода.

## Вариант 2 — Карточки отделов (аккордеон)

Каждый отдел — своя карточка. Клик — раскрытие подотделов и продуктов.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ Отделы и продукты                                              [🔍 Поиск...]   │
│ [+ Отдел] [+ Подотдел] [+ Привязать продукт]                                     │
├─────────────────────────────────────────────────────────────────────────────────┤
│ ┌─ Молоко (12 продуктов) ───────────────────────────────────────── [▼] [✎] [✕] ┐
│ │  ┌─ Чищенка (5) ─────────────────────────────────────── [▼] [✎] [✕] [☑]   │
│ │  │  Молоко 3.2%  л    Кефир 1%  л    Сметана  л    ...                      │
│ │  └─ Сыпучка (3) ─────────────────────────────────────── [▶] [✎] [✕] [☑]   │
│ │  Молоко 3.2% (отдел)  л                                                      │
│ └──────────────────────────────────────────────────────────────────────────────┘
│ ┌─ Овощи (8 продуктов) ───────────────────────────────────────── [▶] [✎] [✕] ┐
│ └──────────────────────────────────────────────────────────────────────────────┘
└─────────────────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: компактно, видна иерархия, быстрый доступ к действиям.  
**Минусы**: при многих отделах — длинная прокрутка.

## Вариант 3 — Улучшенное дерево (текущее + доработки)

Сохраняем дерево, но:
- Убираем колонку «Тип» (и так видно по иконке)
- Группируем кнопки в выпадающее меню «Добавить»
- Карточный стиль для отделов (фон, отступы)
- Бейджи с количеством продуктов

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ Отделы и продукты                                              [ℹ] [Сохранить] │
│ [🔍 Поиск...]                    [▼ Добавить] [+ Отдел] [+ Подотдел] [+ Продукт] │
├─────────────────────────────────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │ 📁 Молоко                                                     [12] [☑]     │ │
│ │   ├─ 📂 Чищенка                                              [5]  [☑]      │ │
│ │   │    ├─ Молоко 3.2%  л                                                   │ │
│ │   │    ├─ Кефир 1%  л                                                      │ │
│ │   │    └─ ...                                                               │ │
│ │   ├─ 📂 Сыпучка                                               [3]  [☑]      │ │
│ │   └─ Молоко 3.2%  л  (отдел)                                               │ │
│ └─────────────────────────────────────────────────────────────────────────────┘
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │ 📁 Овощи                                                      [8]  [☑]     │ │
│ └─────────────────────────────────────────────────────────────────────────────┘
└─────────────────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: минимальные изменения, привычный интерфейс, нагляднее.  
**Минусы**: при 50+ продуктах дерево всё ещё длинное.

## Вариант 4 — Вкладки: «Отделы» и «Продукты»

Две вкладки. «Отделы» — дерево отделов/подотделов (без продуктов). «Продукты» — таблица всех продуктов с колонкой «Отдел».

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ Отделы и продукты                                              [ℹ] [Сохранить] │
├─────────────────────────────────────────────────────────────────────────────────┤
│ [Отделы] [Продукты]                                                              │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Вкладка «Отделы»:                    Вкладка «Продукты»:                         │
│ ┌─────────────────────────────┐     ┌──────────────────────────────────────────┐ │
│ │ 📁 Молоко        [☑] [✎][✕] │     │ Продукт      │ Отдел    │ Ед. │ Действия │ │
│ │   📂 Чищенка     [☑] [✎][✕] │     │ Молоко 3.2%  │ Чищенка  │ л   │ [→] [✕]  │ │
│ │   📂 Сыпучка     [☑] [✎][✕] │     │ Кефир 1%     │ Чищенка  │ л   │ [→] [✕]  │ │
│ │ 📁 Овощи         [☑] [✎][✕] │     │ ...          │          │     │          │ │
│ └─────────────────────────────┘     └──────────────────────────────────────────┘ │
│ [+ Отдел] [+ Подотдел]               [🔍 Поиск] [Фильтр по отделу ▼]            │
└─────────────────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: структура отделов отдельно; продукты — в таблице с сортировкой.  
**Минусы**: переключение вкладок; привязка продукта — выбор отдела в таблице.

## Вариант 5 — Мастер-деталь с панелью действий

Слева — компактный список отделов (без подотделов в основном списке). Справа — выбранный отдел: подотделы + продукты. Сверху — панель быстрых действий.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ [🔍 Поиск]  [+ Отдел] [+ Подотдел] [+ Привязать продукт]            [Сохранить] │
├──────────────────┬──────────────────────────────────────────────────────────────┤
│ Отделы           │ Молоко                                                        │
│ ──────────────── │ ──────────────────────────────────────────────────────────────│
│ ● Молоко (12)    │ Подотделы:                                                     │
│ ○ Овощи (8)      │   📂 Чищенка [5] [☑] [✎] [✕]                                  │
│ ○ Мясо (4)       │   📂 Сыпучка [3] [☑] [✎] [✕]                                  │
│                  │ Продукты отдела:                                               │
│                  │   Молоко 3.2%  л                                              │
│                  │ Продукты подотделов:                                           │
│                  │   Молоко 3.2%  л  (Чищенка)  Кефир 1%  л  (Чищенка)  ...       │
└──────────────────┴──────────────────────────────────────────────────────────────┘
```

**Плюсы**: быстрый переход между отделами, детали справа.  
**Минусы**: продукты разнесены (отдел / подотделы).

## Быстрые улучшения (без смены структуры)

1. **Убрать колонку «Тип»** — иконки уже показывают отдел/подотдел/продукт.
2. **Бейджи количества** — `[12]` рядом с названием отдела.
3. **Карточный стиль** — `setObjectName("card")` для строк отделов, отступы.
4. **Группировка кнопок** — `[▼ Добавить]` с подменю: Отдел, Подотдел, Привязать продукт.
5. **Подсказка на кнопках** — «Добавить отдел», «Добавить подотдел в выбранный отдел» и т.д.
