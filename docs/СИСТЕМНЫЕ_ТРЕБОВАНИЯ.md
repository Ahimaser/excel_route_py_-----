# Системные требования и рекомендации

## Минимальные требования

| Параметр | Значение |
|----------|----------|
| **ОС** | Windows 10 / 11 (64-bit) |
| **ОЗУ** | 2 ГБ |
| **Процессор** | 1 ГГц (x64) |
| **Свободное место** | 500 МБ (для установки exe) |
| **Экран** | 1024×768 пикселей |

## Рекомендуемые характеристики

| Параметр | Значение |
|----------|----------|
| **ОЗУ** | 4 ГБ и более |
| **Процессор** | 2 ядра, 2 ГГц |
| **Свободное место** | 1 ГБ |
| **Экран** | 1366×768 или выше |

## Для работы с большими объёмами данных

При обработке **более 5000 маршрутов** или **более 100 продуктов**:

| Параметр | Рекомендация |
|----------|--------------|
| **ОЗУ** | 8 ГБ |
| **Процессор** | 4 ядра |
| **Диск** | SSD (ускоряет чтение/запись XLS) |

## Дополнительно

- **Python не требуется** — программа поставляется как исполняемый файл (.exe)
- **Права** — запуск от имени пользователя (не требуется администратор)
- **Данные** — хранятся в папке AppData пользователя
