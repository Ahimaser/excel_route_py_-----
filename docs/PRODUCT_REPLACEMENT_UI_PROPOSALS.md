# Варианты реализации окна замены продукта

## Текущее состояние

Диалог «Замена продукта» с:
- выбором продукта для замены (комбобокс);
- блоком «Заменить на»: один комбобокс + чекбокс «+ второй продукт» (при включении — второй комбобокс и спинбокс «Доля на первый» 0–100%);
- режимом: целиком / частично (количество + выбор учреждений);
- списком активных замен.

```
┌─ Замена продукта ─────────────────────────────────────────────────┐
│ Продукт для замены:  [Молоко 3.2% ▼]                               │
│ Всего в маршрутах: 150 л                                           │
│                                                                    │
│ Заменить на:         [Кефир ▼]   ☐ + второй продукт                │
│ ┌─ (при включении) ──────────────────────────────────────────────┐│
│ │ Второй продукт:   [Сметана ▼]   Доля на первый: [50] %          ││
│ └─────────────────────────────────────────────────────────────────┘│
│                                                                    │
│ ○ Заменить целиком во всех маршрутах                                │
│ ○ Заменить частично: [10] л                                         │
│                                                                    │
│ ┌─ Учреждения для замены ──────────────────────────────────────────┐│
│ │ [Выбрать все] [Снять все]                                        ││
│ │ ☐ 001 — всё учреждение (3 адр.)                                  ││
│ │   ☐ ул. Ленина, 1                                                ││
│ │   ...                                                            ││
│ └──────────────────────────────────────────────────────────────────┘│
│                                                                    │
│ Активные замены:                                                    │
│ • Молоко → Кефир + Сметана (50% / 50%)                              │
│                                                                    │
│                    [Очистить замены] [Применить] [Отмена]           │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: компактно, всё на одном экране.  
**Минусы**: при двух продуктах блок «Второй продукт» появляется динамически, доля — отдельный спинбокс.

---

## Вариант 1 — Карточки «1 продукт» / «2 продукта»

Переключатель режима замены: одна карточка или две. Визуально разделены блоки.

```
┌─ Замена продукта ─────────────────────────────────────────────────┐
│ Продукт для замены:  [Молоко 3.2% ▼]   Всего: 150 л                │
│                                                                    │
│ Режим замены:  (●) 1 продукт   ( ) 2 продукта                       │
│                                                                    │
│ ┌─ Заменить на ──────────────────────────────────────────────────┐│
│ │  [Кефир ▼]                                                      ││
│ └─────────────────────────────────────────────────────────────────┘│
│                                                                    │
│ (при выборе «2 продукта»:)                                          │
│ ┌─ Заменить на ──────────────────────────────────────────────────┐│
│ │  Продукт 1: [Кефир ▼]     Продукт 2: [Сметана ▼]                ││
│ │  Распределение:  [====50%====|====50%====]  (слайдер)           ││
│ └─────────────────────────────────────────────────────────────────┘│
│                                                                    │
│ ... (режим целиком/частично, учреждения) ...                        │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: явный выбор «1 или 2», слайдер нагляднее спинбокса.  
**Минусы**: больше переключений.

---

## Вариант 2 — Визуальная схема «A → B» / «A → B + C»

Схематичное отображение замены: стрелка и блоки продуктов. При добавлении второго — появляется ветвление.

```
┌─ Замена продукта ─────────────────────────────────────────────────┐
│                                                                    │
│     ┌─────────┐                    ┌─────────┐                     │
│     │ Молоко  │ ────────────────►  │ Кефир   │                     │
│     │ 150 л   │                    │         │                     │
│     └─────────┘                    └─────────┘                     │
│                                                                    │
│     [+ Добавить второй продукт]                                     │
│                                                                    │
│ (при добавлении:)                                                   │
│     ┌─────────┐         ┌─────────┐                                 │
│     │ Молоко  │ ──────► │ Кефир   │  50%                            │
│     │ 150 л   │    ╲    └─────────┘                                 │
│     └─────────┘     ╲   ┌─────────┐                                 │
│                      └► │ Сметана │  50%                            │
│                         └─────────┘                                 │
│     [====50%====|====50%====]  (слайдер)                           │
│                                                                    │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: наглядно видно «что на что» меняется, ветвление при двух продуктах.  
**Минусы**: занимает больше места, сложнее в реализации.

---

## Вариант 3 — Таблица замен (несколько замен в одном окне)

Окно показывает список замен в виде таблицы. Добавление новой — строка. Редактирование — по двойному клику или кнопке.

```
┌─ Замена продукта ─────────────────────────────────────────────────┐
│                                                                    │
│  Замены:                                                           │
│  ┌──────────────────┬─────────────────────────────┬──────────────┐│
│  │ Заменить         │ На                           │ Действие     ││
│  ├──────────────────┼─────────────────────────────┼──────────────┤│
│  │ Молоко 3.2%      │ Кефир + Сметана (50/50)      │ [✎] [✕]     ││
│  │ Сливки 20%       │ Масло                        │ [✎] [✕]     ││
│  └──────────────────┴─────────────────────────────┴──────────────┘│
│  [+ Добавить замену]                                                │
│                                                                    │
│  (при добавлении/редактировании — раскрывается форма под строкой)    │
│                                                                    │
│                              [Применить] [Отмена]                   │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: несколько замен видны сразу, удобно управлять списком.  
**Минусы**: форма редактирования одной замены может быть тесной.

## Вариант 4 — Мастер (пошаговый)

Пошаговый мастер: шаг 1 — выбор продукта, шаг 2 — на что заменить (1 или 2), шаг 3 — режим и учреждения.

```
┌─ Замена продукта — Шаг 1 из 3 ────────────────────────────────────┐
│                                                                    │
│  Какой продукт заменить?                                            │
│  [Молоко 3.2% ▼]     Всего в маршрутах: 150 л                      │
│                                                                    │
│                                    [Далее →]                        │
└────────────────────────────────────────────────────────────────────┘

┌─ Замена продукта — Шаг 2 из 3 ────────────────────────────────────┐
│                                                                    │
│  На что заменить?                                                   │
│  (●) Один продукт:  [Кефир ▼]                                      │
│  ( ) Два продукта:  [Кефир ▼]  [Сметана ▼]  50% / 50%              │
│                                                                    │
│                         [← Назад]  [Далее →]                        │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: пошагово, не перегружено.  
**Минусы**: больше кликов, нельзя видеть всё сразу.

## Вариант 5 — Компактная строка с «+»

Текущая логика, но оформление в одну строку: «Заменить на: [A ▼] [+ второй]». При нажатии «+ второй» — инлайн появляется «[B ▼] [50%]».

```
┌─ Замена продукта ─────────────────────────────────────────────────┐
│                                                                    │
│  Продукт для замены:  [Молоко ▼]    150 л                          │
│                                                                    │
│  Заменить на:  [Кефир ▼]  [+ второй]                               │
│                                                                    │
│  (при + второй:)                                                    │
│  Заменить на:  [Кефир ▼]  [Сметана ▼]  [50]% на первый              │
│                                                                    │
└────────────────────────────────────────────────────────────────────┘
```

**Плюсы**: минимальные изменения от текущего, компактно.  
**Минусы**: при длинных названиях продуктов строка может не помещаться.

---

## Рекомендация

- **Вариант 5** — ближе всего к текущей реализации, доработать визуально (слайдер вместо спинбокса для доли).
- **Вариант 1** — хороший баланс наглядности и простоты.
- **Вариант 2** — для максимальной наглядности, если окно используется часто и важно визуально понимать замены.
