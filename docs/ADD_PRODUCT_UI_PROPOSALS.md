# Варианты интерфейса добавления продуктов в справочник

Текущая реализация: последовательные диалоги QInputDialog (название → ед. изм. → отдел).

---

## Вариант A: Карточная форма (всё на одном экране)

```
┌─────────────────────────────────────────────────────────────┐
│  Добавить продукт                                         ✕  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  Название продукта                                  │   │
│  │  ┌─────────────────────────────────────────────┐   │   │
│  │  │ Масло сливочное                              │   │   │
│  │  └─────────────────────────────────────────────┘   │   │
│  │                                                     │   │
│  │  Единица измерения                                  │   │
│  │  ┌──────────┐  кг  ▼                               │   │
│  │  └──────────┘                                       │   │
│  │                                                     │   │
│  │  Отдел                                              │   │
│  │  ┌──────────────────────────────┐  Без отдела  ▼  │   │
│  │  └──────────────────────────────┘                   │   │
│  │                                                     │   │
│  │  ☐ Создать вариант (кол-во в 1 шт)  [  0.18  ]     │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│                    [ Отмена ]    [ Добавить ]                │
└─────────────────────────────────────────────────────────────┘
```

**Плюсы:** Все поля видны сразу, можно быстро заполнить. Опционально — чекбокс «Вариант» для создания варианта с кол-вом в 1 шт.  
**Минусы:** Занимает больше места, чем текущий диалог.

---

## Вариант B: Мастер (пошаговый)

```
┌─────────────────────────────────────────────────────────────┐
│  Добавить продукт                                         ✕  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ●─────────○─────────○                                      │
│  1. Название  2. Ед. изм.  3. Отдел                         │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  Название продукта                                   │   │
│  │  ┌─────────────────────────────────────────────┐     │   │
│  │  │ Введите название...                          │     │   │
│  │  └─────────────────────────────────────────────┘     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│                    [ Отмена ]    [ Далее → ]                 │
└─────────────────────────────────────────────────────────────┘
```

**Плюсы:** Понятная пошаговая логика, меньше когнитивной нагрузки.  
**Минусы:** Больше кликов, дольше ввод.

---

## Вариант C: Встроенная панель (inline)

```
┌─ Справочник продуктов ─────────────────────────────────────┐
│  Поиск: [____________]  Отдел: [Все ▼]   [+ Добавить]      │
├────────────────────────────────────────────────────────────┤
│  ┌─ Добавить ─────────────────────────────────────────┐   │
│  │ [Название________] [кг▼] [Без отдела▼]  [ + ]      │   │
│  └─────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────┤
│  Название              │ Отдел    │ Ед.изм │ Действия      │
├────────────────────────┼──────────┼────────┼───────────────┤
│  Масло сливочное (кг)  │ Молоко   │ кг     │ [В отдел] ...  │
│  Молоко (л)            │ Молоко   │ л      │ [Вариант] ...  │
└─────────────────────────────────────────────────────────────┘
```

**Плюсы:** Добавление без модального окна, всё в одном месте.  
**Минусы:** Занимает место над таблицей, при узком окне может быть тесно.

## Вариант D: Разделённая панель (список + форма)

```
┌──────────────────────────────────────────────────────────────────────┐
│  Справочник продуктов                                                │
├─────────────────────────────┬────────────────────────────────────────┤
│  Список продуктов           │  Новый продукт                          │
│  [Поиск_____________]       │  ┌──────────────────────────────────┐  │
│  ┌─────────────────────┐   │  │ Название                          │  │
│  │ Масло сливочное     │   │  │ [________________________]        │  │
│  │ Молоко              │   │  │ Ед. изм. [кг ▼]                   │  │
│  │ Сметана             │   │  │ Отдел    [Без отдела ▼]           │  │
│  │ ...                 │   │  │                                    │  │
│  └─────────────────────┘   │  │ [ Добавить в справочник ]          │  │
│                             │  └──────────────────────────────────┘  │
│  [+ Добавить продукт]      │                                         │
└─────────────────────────────┴────────────────────────────────────────┘
```

**Плюсы:** Форма всегда видна, можно добавлять несколько продуктов подряд.  
**Минусы:** Нужно перестроить layout справочника.

---

## Вариант E: Быстрое добавление + расширенное

```
  Обычный режим (кнопка "+ Добавить продукт"):
  ┌─────────────────────────────────────┐
  │  Название: [_______________]  [✓]   │  ← компактная строка
  │  (ед. изм. и отдел — по умолчанию)   │
  └─────────────────────────────────────┘

  Расширенный режим (ссылка "Подробнее..." или Alt+Enter):
  ┌─────────────────────────────────────┐
  │  Название:     [_______________]    │
  │  Ед. изм.:     [кг ▼]              │
  │  Отдел:        [Без отдела ▼]      │
  │  Вариант:      [  0.18  ] [кг]      │
  └─────────────────────────────────────┘
```

**Плюсы:** Быстрый ввод для типичного случая, детали — по необходимости.  
**Минусы:** Два режима могут запутать.

---

## Рекомендация

| Вариант | Сложность | UX | Когда выбирать |
|---------|-----------|-----|----------------|
| **A**   | Низкая    | ★★★★ | Универсальный, замена текущих QInputDialog |
| **B**   | Средняя   | ★★★  | Если важна пошаговая валидация |
| **C**   | Средняя   | ★★★★ | Если часто добавляют много продуктов подряд |
| **D**   | Высокая   | ★★★★★| При редизайне всего справочника |
| **E**   | Низкая    | ★★★★ | Компромисс скорость/гибкость |

**Оптимальный старт:** Вариант A — одна карточка с полями, без пошаговых диалогов. Минимальные изменения, заметное улучшение UX.
